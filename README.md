#Sobre o Projeto "Turismo pelo Japão"
Este é um projeto desenvolvido durante a Imersão Alura, com o auxílio do Google Gemini, por Samir Morera Morimoto.

#Objetivo
A aplicação funciona como uma base de conhecimento interativa, permitindo ao usuário consultar informações sobre diversos pontos turísticos do Japão de forma rápida e objetiva.

#Como Funciona?
Utilizando HTML para a estrutura, CSS para o design e JavaScript para a interatividade, o site realiza uma busca dinâmica em uma base de dados local (arquivo data.json).

Ao digitar no campo de busca, o sistema filtra e exibe em tempo real os locais correspondentes, apresentando cada um em um cartão com nome, descrição, ano e um link para mais informações.
